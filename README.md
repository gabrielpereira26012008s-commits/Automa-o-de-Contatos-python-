# Automa-o-de-Contatos-python-
Programa em python para automação de adição de números(Contatos) em csv ou xls, para o formato aceito no google contatos (Vcalendar).
-------------------------------------------------------
Módulo nativo paar rodar com expressoões regulares (Biblioteca 're' em python)
é necessario instalar o python 3.13.5 em diante para a últilização do programa.
